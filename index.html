Tropical Day

O ultimo projeto da faculdade

No meu ultimo ano da faculdade, eu decidi fazer um projeto que envolvesse um pouco de tudo, meu foco não foi entregar um jogo completo no final, mas sim me profissionalizar cada vez mais dentro da ferramenta! Dentro dele trabalhei com: Shaders, Animações, SkyBox, Tipos diferentes de colisões, CardBoard, Entre outras coisas. Dentro do jogo o player controla um pirata em uma ilha, e seu objetivo é esplora-la e encontrar os miniGames escondidos!

  • Data: 2018
  • Materia: Oficina 5 e 6
  • Categoria: Puzzle




A View To A Kill

Global Game Jam 2017

Feito utilizando o Tobii EyeX, um dispositivo que consegue identificar para onde na tela o usurio esta olhando com alta precisão. Dentro do jogo você consegue soltar raio-laser pelos olhos, atirando para todo local onde olha, parando apenas quando fecha os olhos. O jogo está disponível para download no link a baixo, mas é necessário o Tobii EyeX para conseguir jogar.

  • Data: Janeiro de 2017
  • Evento: GGJ 2017
  • Categoria: For Fun

Shinob-Z

Um Beat 'em up de Zombies

Após sermos introduzidos à linguagem C++, recebemos o desafio de fazer um jogo usando a openFrameworks, uma toolkit open souce para programação. Resolvemos então tentar fazer um jogo ao estilo dos classicos dos arcades, um jogo Beat 'em up. Com uma premissa simples, o jogador controla um ninja e deve sobreviver as waves de zombies, trocando de personagem quando necessário.

  • Data: Segundo semestre de 2016
  • Materia: Oficina 2
  • Category: Beat 'em Up
